Tos helper - бот, созданный для помощи игрокам Tower of saviors Поддерживает комманды: - /show [id] - показать информацию для карты с id=[id] (id карты написан на каждой карте в самом низу) - /s [id] - показать краткую информацию для карты с id=[id] - /find [name] - найти карту по имени. - /f [name] - то же, что и /find - /report [message] отправить сообщение с описанием ошибки - /poll - создать новое голосование. Комманда должна быть ответом на любое сообщение, для которого будет проводится голосование Информация, которой обладает бот, может быть неточной или неактуальной. Если вы заметили ошибки в работе бота, пожалуйста отправьте /report, указав в сообщении всю необходимую информацию. Создан Виктор[Redvel] для лучшей гильдии!
